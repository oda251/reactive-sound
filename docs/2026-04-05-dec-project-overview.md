# Reactive BGM - プロジェクト概要

## ビジョン

キーボード入力や環境情報に応じてリアルタイムにBGMが変化する、バックグラウンド常駐型デスクトップアプリケーション。

## 初期スコープ

- バックグラウンド常駐でキーボード入力を検知
- 入力パターンに応じて BPM・音色がリアルタイムに変化
- ミニマルな GUI でパラメータの確認・手動調整が可能

## 将来の展開

- 環境連動（天気、時間帯、CPU負荷など外部データによる変化）
- プリセット管理（シーン切り替え）
- MIDI コントローラー対応

## ターゲット

個人ツール。自分自身が日常的に使うBGMツールとして開発する。

## 技術スタック概要

| レイヤー | 技術 | 役割 |
|---------|------|------|
| 音声合成 | SuperCollider (scsynth) | シンセサイザー・音声エンジン |
| 作曲/パターン | TidalCycles (GHCi) | パターン記述・シーケンシング |
| アプリケーション | Rust | プロセス管理・入力検知・通信 |
| GUI | egui | パラメータ表示・スライダー/ノブ操作 |
